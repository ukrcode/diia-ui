# Contributing

Here you can find details how to contribute to this package.

## GIT Commits

We suggest following [conventional commits convention](https://www.conventionalcommits.org/en/v1.0.0/)

### Possible commit prefixes:

- feat: 
- fix
- build:
- chore:
- ci:
- docs:
- style:
- refactor:
- perf:
- test:
